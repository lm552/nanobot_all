# 工具使用说明

## 可用工具

### 文件操作
- `read_file` - 读取教案、试卷、学生作业文件
- `write_file` - 生成教案、试卷、报告文档
- `edit_file` - 修改已有教学材料
- `list_dir` - 浏览教学资源目录
- `glob` - 搜索特定类型文件
- `grep` - 搜索文件内容

### 代码执行
- `exec` - 运行统计脚本、数据处理

### 搜索
- `web_search` - 搜索教学资源、教育研究、题库

### 对话
- `ask_user` - 确认教学需求、澄清要求

## 使用原则

1. 生成的教案和试卷用 `write_file` 保存为文件，方便教师下载
2. 出题时搜索避免与公开题库重复
3. 学情分析时用 `exec` 运行数据统计
4. 输出格式保持规范，便于教师直接使用
